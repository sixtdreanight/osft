## 1. 项目概述

### 1.1 核心方法

OSFT (Orthogonal Subspace Fine-Tuning)：对预训练 GAN 权重矩阵做 SVD 分解，将主子空间冻结，只微调残差子空间。

```
预训练权重 W (79M params)
  ↓ SVD (τ=0.80, 保留 80% 能量)
┌─────────────────────┬─────────────────────┐
│   Wr (冻结)          │   dW (可训, 523K)    │
│   拓扑结构知识        │   物理约束适配        │
│   保护 IOU, CKA      │   降低 MSE, 适应工况   │
└─────────────────────┴─────────────────────┘
```

### 1.2 环境

- **GPU**: NVIDIA RTX 5070 Ti Laptop (12GB), 利用率 78%, 温度 75°C, 功率 127W
- **Python**: 3.14, PyTorch 2.x, Solidspy 1.1
- **模型**: TopologyGAN (unet, gf_dim=64, 79M 参数, 62M 可分解)
- **数据**: Cantilever beam 拓扑优化, 64×128 分辨率, 7 通道

### 1.3 两种数据集

| 数据集 | 来源 | 样本数 | 大小 | 特点 |
|--------|------|--------|------|------|
| 合成 (synthetic) | 随机平滑场生成 | 500 | 110 MB | 无真实结构, 均匀分布 |
| 真实 (cantilever) | MEGA下载, SIMP求解 | 528 | 115 MB | 真实拓扑优化, 含FEM结果 |

## 5. 理论交叉分析

### 5.1 OSFT 与持续学习

```
GAN 微调 = 持续学习
  任务 A (预训练): 生成有效拓扑
  任务 B (物理约束): 合规性 + 体积分数

OSFT = EWC + OWM
  SVD 能量截断 ≈ Fisher 信息对角近似 (EWC)
  冻结 Wr ≈ 更新投影到零空间 (OWM)
```

### 5.2 可交叉理论

| 领域 | 理论 | 应用方向 | 优先级 |
|------|------|---------|--------|
| 持续学习 | OWM | FIX2 理论基础 (已做, 负结果) | ✅ |
| 拓扑数据分析 | 可微持久同调 | 修复 β1=0, L_topo | 🔴 |
| 持续学习 | EWC Fisher | 替代能量截断选 τ | 🟡 |
| PEFT | PSOFT | 方法对比基线 | 🟡 |
| PEFT | SC-LoRA | 数据驱动 τ | 🟢 |
| PEFT | PESO-LoRA | OSFT 理论定位 | 🟢 |
| 物理信息 | PI-Diffusion | 升级物理损失 | 🟢 |

**结论**: 显式投影 < 优化器隐式约束。OSFT 原始设计最优。

## 7. 论文叙事建议

### 7.1 三贡献

1. **理论**: GAN 微调 = 持续学习。OSFT = EWC + OWM 特例
